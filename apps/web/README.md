## Visual Styles

When creating a video (Blog, Text, or Talk mode), you can choose a **Visual Style** that controls how Amazon Nova Reel renders the footage and how the Nova Canvas thumbnail is generated:

| Style | Look |
|-------|------|
| 🎬 Cinematic | Realistic live-action footage (default) |
| 🎨 Cartoon | 2D flat-design animation |
| 🌸 Anime | Cel-shaded Japanese animation style |
| 🧱 Claymation | Stop-motion clay aesthetic |
| 🖌️ Watercolor | Hand-painted illustration |

The selection flows from the UI through the API and Step Functions pipeline into the render worker, which injects the matching style descriptor into every Nova Reel scene prompt.

You can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
